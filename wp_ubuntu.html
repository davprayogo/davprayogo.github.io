Tugas

Membuat website wordpress dengan database terpisah

Topologi:

Alat:
virtualBox
2 VM ubuntu server yang sudah di setup

Langkah-langkah
1. Siapkan 2 VM ubuntu server untuk menjadi databases dan webserver

2. Setting ip statis yang menggunakan adapter host-only adapater. Dengan menjalankan comment dibawah:
"nano /etc/netplan/00-isntaller-config.yaml"

Kemudian jalankan comment ini : sudo netplan apply
berfungsi unutk mengecek sintak yang kita buat apakah ada error jika tidak maka akan tampil seprti gambar diatas.
Lakukan di kedua VM agar bisa saling terhubung.
Dan jangan lupa lakukan update dan upgrade pada kedua VM dengan comment:
"sudo apt update"
"sudo apt upgrade"

Setup VM web server

Menginstall apache2 di ubuntu server
comment "sudo apt install apache2 -y"

Menginstall mysql client
comment "sudo apt install mysql-client -y"

Menginstall php dan php-mysql
Comment "sudo apt install php php-mysql"

Untuk mengcek apakah service apache2 sudah run apa belum anda bisa menggunakan comment
"sudo systemctl status apache2"
Dan jika ingin merestart service apache2 anda bisa menggunakan comment.
“sudo systemctl restart apache2”
Dan unutk mengecek tampilan webnya seperti apa kalian bisa buka di browser dengan mengetikan.
http://ip_webserver/
Maka akan muncul tampilan seprti ini sebagai default apache2 page

Setup Database Server

Menginstall mysql server
Comment “sudo apt install mysql-server -y”

Menginstall php dan php-mysql
comment “sudo apt install php php-mysql”

Jika sudah untuk mengecek service mysql anda bisa menggunakan commant.
“sudo systemctl status mysql”
Dan unutk merestart service anda bisa menggunakan commant.
“sudo systemctl restart mysql”

konfigurasi mysql secure

sebelum kita konfigurasi mysql secure nya kita harus mengrubah password root dari msyql server terlebih dahulu. Commant:
"sudo mysql"
“ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY;”
Jika sudah ketik “exit” Jika sudah ketik “exit” dan kemudian jalankan comment:
“mysql_secure_installation”
Dan kemudian anda bisa ikutin pada gambar dibawah:


Jika sudah makan kita akan membuat user dan database kemudian memberikan hak akses nya juga Pertama kita harus login terlebih dahulu ke myql nya dengam comment:
“ mysql –u root –p ” Jika sudah kita akan membuat user baru dan database untuk kita gunakan kebada web server dan database untuk website kita nanti dan memberi perivilage ke pada user yang akan kita buat nanti ke database untuk website kita nanti. Commant:
“CREATE USER ‘user_name’@’ip_webserver’ IDENTIFIED BY ‘passwourd_anda’;”
“CREATE DATABASE nama_database;”
“GRANT ALL ON nama_database.* TO ‘user_name’@’ip_webserver’“
“FLUSH PRIVILEGES;”

Selanjutnya kita akan mengkonfigurasi bind-address commant:
“sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf“
Jika sudah save file nya dan restart service mysql.
Untuk pengecekan apakah sudah terhubung kita bisa mencoba login ke mysql menggunakan user yang telah kita buat sebelumnya di webserver. Commant:
“sudo mysql –u user_name –h ip_database -p“

Menginstall wordpress di webserver

Pertama kita harus download file wordpress nya dengan commant:
“cd /tmp && wget https://wordpress.org/latest.tar.gz“

Selanjutnya kita extrak file wordpress Commant:
“ tar -xvf latest.tar.gz ”

Jika sudah selesai di extrak maka akan muncul folder wordpress di didalam folder /tmp. Selanjutnta kita copy folder wordpress tadi ke folder /var/www/html. Commant:
“ sudo cp -R wordpress /var/www/html/ ”
Kemudian jika sudah maka kita akan mengubah hak pemiliki direktori menjadi www-data:www-data. Commant:
“ sudo chown -R www-data:www-data /var/www/html/wordpress/ ”
Selanjutnya kita akan mengubah hak akses direktori menjadi 755 yang berarti akses baca dan eksekusi untuk semua orang dan juga akses write untuk pemilik file. Commant:
“ sudo chmod -R 755 /var/www/html/wordpress/ ”
Kemudian kita akan membuat sebuah direktori uploads di direktori /var/www/html/wordpress/wp_content/. Commant:
“ sudo mkdir /var/www/html/wordpress/wp-content/uploads ”
Dan juga mengubah hak pemiliik direktori menjadi www-data:www-data. Commant:
“ sudo chown -R www-data:www-data /var/www/html/wordpress/wp-content/uploads/ ”

Jika sudah kita bisa restart service apache2,
Untuk pengcekan bisa kita lakukan di browser dengan mengetikan:
“ http://ip_webserver/wordpress ”
Dan jika berpahasil penginstallannya maka akan muncul tampila seprti ini pada browser anda.
