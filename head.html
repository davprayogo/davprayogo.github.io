Install Wordpress Menggunakan xampp

XAMPP, untuk menyediakan PHP environment, web server dan MySQL server. Download aplikasi ini di: https://www.apachefriends.org/download.html . Pada matakuliah ini, kita akan menggunakan PHP versi terbaru yaitu PHP8 sehingga pilih versi XAMPP dengan awalan 8.x saat proses download. Lakukan instalasi hingga selesai.
Setiap memulai proses pengembangan web, pastikan XAMPP pada bagian Apache dan MySQL server telah dijalankan seperti tampilan berikut.
